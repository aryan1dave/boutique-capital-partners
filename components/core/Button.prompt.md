BCP's button in all four site treatments; use `primary` for the main CTA on light or navy, `outline` on navy panels, `cream` on teal/photo backgrounds, `text` for inline "Read article →" CTAs.

```jsx
<Button arrow href="#contact">Partner with us</Button>
<Button variant="outline" arrow>Discuss with us</Button>
<Button variant="cream">Read the press release</Button>
<Button variant="text" arrow>Read article</Button>
```

Notes: hover states (darken/lift/glow, sliding arrow) are built in. `tertiary` is the small `#E8E4DC` button used inside cream callout cards.
